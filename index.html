Links

Default links are boring and instantly change state when hovered.

Default html link

Yawn. Adding a half second transition takes one line of code and makes the link more interactive.

Transition on hover


.links .link-transition {
  -webkit-transition:color .5s ease-in;
  -o-transition:color .5s ease-in;
  -moz-transition:color .5s ease-in;
  transition:color .5s ease-in;
}
.links .link-transition:hover {
  color:#FF0084;
}

Run these through Autoprefixr to automatically prefix your style rules.

Let's take it a step further.

Many links are underlined when hovered, like so:

Default underline.

But we can use pseudo elements to animate the underline.

Animate the underline

Underline from center

Underline from right

Animating the underline from left.


.animate-underline {
    position: relative;
    transition:all .5s ease-out;
    &:hover {
        color: $pink;
    }
    &:after {
        content:"";
        position:absolute;
        left:0;
        bottom:0;
        border-bottom:5px solid $lightblue;
        width:0%;
        transition: all .5s ease-out;
    }
    &:hover:after {
        border-bottom:5px solid $pink;
        width:100%;
    }
}

Animating the underline from center.


.underline-center {
    position: relative;
    transition: all .5s ease-out;
    &:hover {
        color:$pink;
    }
    &:after {
        content: "";
        position:absolute;
        left:50%;
        right:50%;
        bottom:0;
        height:0px;
        background-color:$lightblue;
        transition: all .5s ease-out;
    }
    &:hover:after {
        left:0px;
        right:0px;
        background-color:$pink;
        height:4px;
    }
}
