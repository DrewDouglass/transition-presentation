We can improve a websites image by taking a few minutes to add simple transitions and effects to interactive elements.

Kevin often refers to this as the last 5% of the website and it makes a huge difference to clients and users first impression of your brand.

Links

Default links are boring and instantly change state when hovered.

Default html link

Yawn. Adding a half second transition takes one line of code and makes the link more interactive.

Transition on hover


.links .link-transition {
  -webkit-transition:color .5s ease-in;
  -o-transition:color .5s ease-in;
  -moz-transition:color .5s ease-in;
  transition:color .5s ease-in;
}
.links .link-transition:hover {
  color:#FF0084;
}

Run these through Autoprefixr to automatically prefix your style rules.

Going forward, I will be using SASS for brevity in the code examples.

Animating the underline from left.


.animate-underline {
    position: relative;
    transition:all .5s ease-out;
    &:hover {
        color: $pink;
    }
    &:after {
        content:"";
        position:absolute;
        left:0;
        bottom:0;
        border-bottom:5px solid $lightblue;
        width:0%;
        transition: all .5s ease-out;
    }
    &:hover:after {
        border-bottom:5px solid $pink;
        width:100%;
    }
}

Animating the bracket navs with transitions and pseudo elements.


.nav {
    width:100%;
    list-style-type: none;
    padding-left:0;
    margin-left:0;
    background-color:$darkblue;
    padding:5% 2%;
    text-align:center;
    li {
        display:inline-block;
        font-size:30px;
        margin:0 1%;
        text-transform: uppercase;
        a  {
            color:$white;
        }
    }
}
.nav-bracket {
    a {
        transition: all 0.5s ease-out;
        &:hover {
            color:$white;
        }
        &:before {
            display:inline-block;
            transition:all .5s ease-out;
            content: "[";
            transform: translateX(20px) rotate(0deg) scale(1);
            opacity:0;
            margin-right:10px;
        }
        &:after {
            display:inline-block;
            transition:all .5s ease-out;
            content: "]";
            transform: translateX(-20px) rotate(0deg) scale(1);
            opacity:0;
            margin-left:10px;
        }
        &:hover:before, &:hover:after {
            transform:translateX(0px);
            opacity:1;
        }
    }
}

"Sword" Navigation Example


.nav-swords {
	background-color: $gold;
	li {
		a {
			position: relative;
			display:inline-block;
			padding:0px 20px;
			height: 45px;
			line-height: 45px;
			margin: 15px;
			outline: none;
			color: #fff;
			text-decoration: none;
			text-transform: uppercase;
			font-weight: 400;
			text-shadow: 0 0 1px rgba(255,255,255,0.3);
			&:before, &:after {
				transition:all .3s;
				width:45px;
				height:2px;
				opacity:0.2;
				background-color:$white;
				position: absolute;
				box-sizing: border-box;
			}
			&:before {
				top:0;
				left:0;
				transform: rotate(90deg);
				transform-origin: 0 0;
				content:"";
			}
			&:after {
				bottom:0;
				right:0;
				transform:rotate(90deg);
				transform-origin: 100% 0;
				content:"";
			}
			&:hover:before {
				left:50%;
				transform:rotate(0deg) translateX(-50%);
				opacity:1;
				width:50%;
			}
			&:hover:after {
				right:50%;
				transform:rotate(0deg) translateX(50%);
				opacity:1;
				width:50%;
			}
		}
	}
}

It conveys to the user that the site is one of quality and attention to detail, it stands out from the sea of others.
